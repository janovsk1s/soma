# Screenshots and the Paka consistency check

Before each milestone ships, place fresh Soma screenshots side by side with Paka's
(`paka/docs/screenshots/`) and check that a stranger would assume the two apps are one
product family:

- pure black background, white Akkurat type, grey secondary text — no accent colors;
- 16 sp centered screen title, 30 sp lowercase settings rows with dim trailing state;
- five rows per page with the right-edge line indicator;
- drawn monochrome icons (gear, plus, back arrow) with Paka's exact geometry.

Device photos of the app running on a Light Phone III belong in this folder, named
after the screen they show (`home.jpeg`, `todos.jpeg`, `settings.jpeg`, `browser.jpeg`),
mirroring Paka's convention.
